A serverless, progressive web application (PWA) that lists upcoming events from a Google Calendar. Events can be sorted by city or preferred display number. Information about the type of event and the number of events per city is summarized via data visualization components.

This project uses AWS and React with both test-driven (TDD) and behavior driven (BDD) development. Calendar events are fetched via the Google Calendar API and data visualizations are created with Recharts.
